# NHS Cancer Bed Availability Analysis

This project analyzes the availability of hospital beds for cancer treatment in NHS Scotland using Microsoft Excel.

The study focuses on understanding trends in cancer treatment waiting times and bed occupancy across different health boards in Scotland. The project applies forecasting techniques and optimization models to support healthcare planning and resource allocation.

## Objectives

- Analyze cancer treatment waiting time data in NHS Scotland
- Examine bed occupancy rates in the Medical Oncology departments
- Apply time series forecasting techniques to predict future demand
- Use linear optimization to improve resource allocation and staff planning

## Methods Used

The analysis includes the following methods:

- Data cleaning and preparation
- Trend analysis using Excel
- Time series forecasting
- Linear optimization modelling
- Visualization of healthcare data trends

## Dataset

The datasets used in this project include:

- Cancer waiting time data across NHS Scotland health boards
- Hospital bed occupancy data for the Medical Oncology departments

The datasets were cleaned and filtered in Microsoft Excel to focus on relevant departments and time periods.

## Tools Used

- Microsoft Excel
- Time Series Forecasting
- Linear Programming / Optimization
- Data Visualization

## Key Insights

- Some health boards experienced reduced performance in meeting the 62-day cancer treatment target.
- Bed occupancy dropped during 2020 due to the impact of the COVID-19 pandemic.
- Forecasting models helped identify future demand patterns.
- Optimization models can support better allocation of hospital staff and beds.
